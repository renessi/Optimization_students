assignment_students.py 
- основной файл с кодом. Запускать в python без аргументов.
  Внутри файла много комментариев. 
  Список необходимых библиотек указан в начале файла
  
limits_all_4.txt 
- данные с лимитами набора на кафедру для равномерного случая (все по 4)

limits_some_7.txt
- данные с лимитами набора на кафедру для такого же случая,
  но с увеличенным лимитом до 7 для двух самых популярных направлений
  
priorities.txt
- Численное значение приоритетов при оптимизации. 
  В assignment_students.py есть детальные комментарии об этом.
  
проекты.xlsx
- исходный файл с данными

проекты_limits_all_4_результат.xlsx
- результат работы на данных проекты.xlsx и limits_all_4.txt 
  Генерируется автоматически заново при запуске assignment_students.py 

проекты_limits_some_7_результат.xlsx
- результат работы на данных проекты.xlsx и limits_some_7.txt 
  Генерируется автоматически заново при запуске assignment_students.py 

консоль.txt
- Пример вывода программы assignment_students.py 
  с дополнительной отладочной информацией
